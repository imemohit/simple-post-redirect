This plugin allows you to make simple redirects of single pages of any custom post type to any url.

== Description ==
Using this plugin you can redirect any page of any post type to a url on your own or external website. This plugin adds an option metabox to single pages of all default and custom post types.

== Installation ==


    Upload Simple Page Redirect to the /wp-content/plugins/ directory

    Activate the plugin through the ‘Plugins’ menu in WordPress

    Edit any page and add destination URL to the option metabox on the top right.
	
	For removing a redirect, simply blank out the field.
